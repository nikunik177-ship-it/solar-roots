# 🌿 SolarRoots — Smart Solar Irrigation Dashboard

SolarRoots is a modern, responsive, and highly aesthetic smart irrigation dashboard. It allows you to monitor soil health, ambient conditions, tank levels, and solar yield in real-time, designed specifically for rosemary and lemon grass cultivation.

---

## ✨ Features
- **🌿 Dual Plant Profile Monitoring**: Pre-configured profiles and telemetry thresholds for Rosemary and Lemon Grass.
- **☀️ Solar telemetry & Battery status**: Live updates of solar panel output power and system battery percentage.
- **📈 Real-Time Rolling Charts**: Dynamic, smooth, canvas-rendered stream charts powered by Chart.js for Light Intensity, Air Moisture, Soil Moisture, Water Flow, and Tank Level.
- **🎛️ Manual Override & Simulation Mode**: Inject custom sensor data (Light, Air, Soil, Flow, Tank, pH) into the live stream to test hardware limits or run manual irrigation cycles.
- **📅 Interactive Schedule & Active Alerts**: Keep track of upcoming waterings and view system logs/alerts in an elegant sidebar drawer.
- **📓 Planter's Journal**: Keep a digital log of observations, treatments, and growth stages directly inside the dashboard.
- **🎨 Curated Modern Themes**: Fully dynamic themes (Sage-Purple, Rosemary-Green, LemonGrass-Yellow) with glassmorphism cards and micro-animations.

---

## 🛠️ Stack & Libraries
- **Frontend Core**: HTML5, Vanilla JavaScript (ES6+), CSS3 (Custom Properties & Animations)
- **Charts**: [Chart.js](https://www.chartjs.org/) (via CDN)
- **Fonts**: Cormorant Garamond & DM Mono (via Google Fonts)

---

## 🚀 How to Run Locally
Simply open the `index.html` file in any modern web browser! No installation or local server is required.

---

## 🌐 Hosting on GitHub Pages
This project is configured for **1-click hosting** via GitHub Pages:
1. Create a repository on GitHub.
2. Push this folder to your repository.
3. Go to **Settings > Pages** on your GitHub repository.
4. Select `main` branch and `/ (root)` folder, then click **Save**.
5. Your dashboard will be live at `https://<your-username>.github.io/<repository-name>/`!
